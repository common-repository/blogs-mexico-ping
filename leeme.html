Introduccion

Blogs Mexico Ping, es un plugin diseñado para funcionar bajo WordPress 2.0.x, el cual sirve para enviar pings de manera automatica cada vez que publiquemos una nueva entrada o editemos una previa.

Requerimientos

  • PHP 4.1 (o mayor)
  • Soporte para sockets
  • WordPress 2.0.x

Configuracion e Instalacion

  1. Desempaca el Archivo en un directorio vacio
  2. Abre con cualquier editor el archivo "blogs_mexico_ping.php"
  3. En las lineas 42 y 45, se encuentran las variables a cambiar, que corresponden al titulo y a la URL respectivamente, le recordamos poner exactamente la informacion como la registro.
  4. Subir el archivo guardado al directorio "/wp-content/plugins/".
  5. Entra a tu panel de administracion o dashboard y activa el plugin
  6. Listo ahora cada vez que publiques o edites una entrada, sera notificado de manera automatica blogsmexico.

FAQ's

  • ¿Como se que mi servidor cuenta con soporte para sockets?
    R: Se hace sacando la informacion de PHP, atravez de la funcion phpinfo() crea un archivo que contenga algo como: "<?php phpinfo();?>"
    una vez que lo ejecutes checa que en el modulo de sockets el soporte se encuentre en modo "enabled"
  • ¿Por que no aparezco en la lista de actualizados?
    R: Primero checa que se encuentre activado el plugin, segundo checa el soporte de sockets, tercero checa que el titulo y url sean los correctos, si aun asi no sirve te pido me contactes para tratar de solucionar tu problema y ver si es algun fallo del software
  • ¿Por que hiciste el plugin?
    R: Por que me daba flojera ir cada vez y de manera manual rellenar el formulario por cada nota nueva, asi basicamente siguo manteniendo una pseudo funcion ping, como la de wordpress pero especialmente diseñada para blogsmexico.com
  • ¿Piensas Mejorar el plugin?
    R: Todo depende del rumbo que lleve el plugin
